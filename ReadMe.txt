# Step to run
    1. Navigate in terminal to app folder
    2. Type `npm install` in terminal to install dependencies
    3. Type `npm start` in terminal to run the application
        - New browser tab will open with app running

# State Management
- Using Redux to control global app State

# Data retrieval 
- Using axios for api calls

# Component Architecture
- App
    - User List 
        - Search
        - Select
        - User List Item

    - User View
        - User Details
